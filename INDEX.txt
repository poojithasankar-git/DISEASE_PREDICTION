BANANA DISEASE CLASSIFIER - INDEX
=================================

DEPLOYMENT (RENDER)
- Build Command: pip install -r requirements.txt
- Start Command: gunicorn --workers 1 --timeout 120 --bind 0.0.0.0:$PORT app:app
- Optional env var: BACKUP_SVC (external verifier API key)

MODEL FILES
- models/banana_model.h5
- models/metadata.json

API ENDPOINTS
- GET /health
- GET /info
- POST /predict

NOTES
- Inference-only (no training on deploy)
- No Kaggle credentials required
